# idk

A fun CLI tool for when you don't know what to do! Get decisions, activity suggestions, and motivational quotes right from your terminal.

## Installation

```bash
# Clone the repository
git clone <your-repo-url>
cd idk

# Make the script executable
chmod +x index.js

# Optional: Link it globally
npm link
```

## Usage

```bash
# Get a quick yes/no decision
idk

# Get advice on a specific decision
idk decide "Should I learn Rust?"

# Get a random activity suggestion
idk do

# Get a motivational quote
idk quote

# Show help
idk help
```
